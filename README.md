# arun4by4.github.io

Personal GitHub Pages site hosted at [arun4by4.github.io](https://arun4by4.github.io).

Built with [Jekyll](https://jekyllrb.com/) using the [Cayman theme](https://github.com/pages-themes/cayman). Content is authored in Markdown and published automatically on every push to `main`.

## Local Preview

```bash
gem install bundler jekyll
jekyll serve
```

Open `http://localhost:4000`.
