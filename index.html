About Me

I am a Senior Software Engineer with over 15 years and 6 months of enterprise software development experience, specialising in building scalable, maintainable, and cloud-ready systems.

My core expertise spans Java / J2EE, Spring Boot microservices, and cloud-native architecture on Google Cloud Platform. I take full SDLC ownership — from initial requirements analysis and architecture design through to deployment, monitoring, and ongoing operations.

I have delivered projects under both Agile and Waterfall methodologies, and I am comfortable leading technical discussions, mentoring team members, and driving quality through code reviews and static analysis tooling.

More recently I have embraced AI-assisted development, sharpening my Prompt Engineering skills with tools such as GitHub Copilot and Cursor to accelerate delivery without sacrificing code quality.

I improved processing efficiency across multiple engagements by implementing Microservice architecture patterns, and I consistently deliver reusable, robust code using Spring MVC principles and clean-architecture guidelines.
